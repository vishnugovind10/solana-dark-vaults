# Code of conduct

## Standard

Contributors must keep discussion technical, specific, and respectful. Harassment, discriminatory language, personal attacks, deliberate intimidation, publishing private information, and sustained disruption are not acceptable. Good-faith critique of code, assumptions, and evidence is welcome; critique people are not.

## Enforcement

Report conduct concerns privately through the repository owner's GitHub profile contact channel. The maintainer may edit or remove contributions, warnings, comments, issues, or participation rights when behavior violates this standard. Enforcement decisions should consider context, severity, repetition, and risk to affected people.

This policy applies in repository spaces and when representing the project in public community spaces.
