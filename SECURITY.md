# Security policy

This repository is unaudited research software. Do not deploy it with value.

Report vulnerabilities through GitHub's private vulnerability reporting for this repository. Include the affected commit, threat scenario, reproduction, impact, and any suggested mitigation. Do not open a public issue for an unpatched vulnerability.

The maintainer will acknowledge a complete report within seven days, assess scope, and coordinate disclosure after a fix or documented mitigation. There is no bug bounty or guaranteed response SLA.

In scope: Anchor account validation, arithmetic, callback authorization, replay handling, cryptographic misuse, secret persistence, and dependency compromise. Out of scope: production deployment support, social engineering, denial of service against third-party infrastructure, and vulnerabilities in mocked external protocols that cannot affect this repository's code.
